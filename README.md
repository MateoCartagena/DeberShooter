## Introducción

En el proyecto actual, estamos trabajando en el desarrollo de un juego tipo shooter con la plataforma Unity. Siguiendo las instrucciones de nuestro profesor, estamos implementando un sistema de disparos utilizando Raycast para interactuar con objetos específicos y crear una experiencia de juego básica y emocionante. En este juego, tu objetivo es disparar a cubos y esferas, con la particularidad de que los cubos requieren dos disparos para ser eliminados, mientras que las esferas necesitan tres. Para llevar un seguimiento de tu progreso, hemos incorporado un marcador que te muestra cuántos cubos y esferas has destruido. La función "destroy" de Unity se utiliza para eliminar estos objetos una vez que se han alcanzado los disparos necesarios.
